# /refactor

Use the installed coding-agent skills to handle this as a safe refactor.

Target:
$ARGUMENTS

Required workflow:
1. Use refactoring skill.
2. Preserve behavior.
3. Identify code smells and coupling.
4. Create staged refactor plan.
5. Do not mix feature work with refactor.
6. Use testing skill before and after changes.
7. Use code-review after implementation.
8. Store durable refactor decisions with Mem0 CLI only if useful for future sessions.

Output:
- Current problems
- Safe refactor stages
- Files affected
- Behavior-preservation tests
- Rollback plan
