# Good Practices Recipe

Use this as an optional repo-health prompt. Do not install it as a default alias unless the team opts in.

Prompt:

```text
Review this repo for the five highest-leverage improvements.

Focus on:
- developer onboarding
- verification commands
- CI and Docker
- dependency hygiene
- security defaults
- agent instructions
- documentation gaps

Do not refactor. Recommend smallest useful next steps with verification commands.
```
