# UI Recipe

Use this as an optional prompt recipe when frontend quality matters. Do not install it as a default alias unless the team opts in.

Prompt:

```text
Design or improve this UI:
<task>

Focus on:
- user workflow and intent
- responsive desktop/mobile behavior
- loading, empty, error, and success states
- accessibility basics
- text overflow and layout stability
- Playwright validation

Before coding, describe the UX intent, layout approach, states, and verification plan.
```
